# Midas Landing Page

A modern landing page for Midas, built with React and Tailwind CSS.

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/yourusername/midas-landing-page.git
```

2. Install dependencies
```bash
cd midas-landing-page
npm install
```

3. Start the development server
```bash
npm start
```

4. Build for production
```bash
npm run build
```

## Technologies Used
- React
- Tailwind CSS
- Lucide Icons

## Project Structure
- `/src/components`: Reusable UI components
- `/src/pages`: Page components
- `/public`: Static assets
